AlignOntology

This code can be used to align any two ontologies where terminal nodes are uniquely identified by their names. Origionally, this code was created to align pairs of gene ontologies for "A gene ontology inferred from molecular networks" doi:10.1038/nbt.2463. The algorithm is described in this paper - please cite this paper if you use this code. Feel free to contact the author (email below) with any questions.
